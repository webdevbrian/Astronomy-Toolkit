How to use

Controls

  • Rotate - Left Click
  • Pan - Right Click
  • Zoom - Scroll or Middle Click
  • Fullscreen mode - F11
  • Increase Speed - Right Arrow
  • Decrease Speed - Left Arrow

Options

  • Speed - speed of the simulation in centuries/second
  • Pause/Start - pause or start the simulation
  • Show Belts - toogle asteroid belts
  • Show Skybox - toogle milky way skybox
  • Follow - select object to follow
  • Designation - name of the body you want to search in database(it can be an asteroid e.g. Ceres, Juno, Pallas or a commet e.g. 1P/Halley, 67P/Churyumovâ€“Gerasimenko, C/1995 O1)
  • Search - Start search
